Data Warehousing Project

Project: Building a Modern SQL Data Warehouse

Tools & Technologies: SQL Server, ETL, Git, Notion, Draw.io, CSV, Data Modeling

  • Designed and implemented a modern data warehouse using the Medallion architecture (Bronze, Silver, Gold layers) for ERP and CRM data integration.
  • Built end-to-end ETL workflows in SQL Server to extract, clean, transform, and load data from CSV sources.
  • Applied data normalization, cleansing, and business rule-based transformations to produce analytics-ready models.
  • Defined and enforced naming conventions, schema structures, and full-load strategies to ensure maintainability and consistency.
  • Planned the entire project using Notion and maintained codebase in a structured GitHub repository with documentation.
  • Delivered a fully functional, production-grade data architecture optimized for business intelligence and analytics use cases.
